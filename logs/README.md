# Logs directory

Thư mục này dùng để nộp minh chứng thực tế của sinh viên.

Ví dụ file có thể đặt ở đây:
- `run-output.txt`
- `run-output-tamper.txt`
- `run-output-wrong.txt`
- `run-output-padding.txt`
- `run-output-ds-sample.txt`
- ảnh chụp màn hình hoặc log test
